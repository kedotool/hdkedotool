⚙️ Hướng Dẫn Setup Tool Yêu Cầu Phiên Bản Python3.12 Nhé

🎉Lưu Ý: Nếu Bạn Gặp Lỗi Dạng No Module Name 'jj đó' thì bạn chỉ cần nhập pip install jj đó

🎉Lưu Ý: Ví Dụ No Module Name 'requests' thì bạn nhập pip install requests là được rồi chạy lại tool

📌 Cài Đặt Trên Android (Termux)

termux-setup-storage && apt update && apt upgrade && pkg install python

📌 Cài Đặt Trên Cloud Shell GG

Copy link này vào trình duyệt https://shell.cloud.google.com/
Bấm mở terminal lên rồi dán lênh này vào: sudo apt install git wget -y && wget https://raw.githubusercontent.com/chhhgggg/h/refs/heads/main/toolkedo.py && pip install bs4 requests && pip install webdriver_manager && cd && python toolkedo.py

📌 Cài Đặt Trên iOS (iSH) Nên Sử Dụng Shell GG nhé

apk update && apk add python3 py3-pip

📌 Cài Đặt Trên PC

python -m pip install --upgrade pip && pip install requests colorama bs4 selenium
